Aqui está o passo a passo para publicar esse site estático gratuitamente usando Netlify ou Vercel:

Como publicar seu site no Netlify

  1. Crie uma conta no Netlify.
  2. Crie uma nova pasta no seu computador, coloque o arquivo acima com o nome index.html.
  3. (Opcional) Adicione também suas imagens na pasta (se não usar links externos).
  4. Acesse o painel do Netlify e clique em “Add new site” > “Deploy manually”.
  5. Arraste a pasta do projeto para a área de upload (“Drag and drop your site folder here”).
  6. Aguarde o deploy. Netlify vai te fornecer um link para seu site publicado.

Como publicar seu site no Vercel

  1. Crie uma conta no Vercel com GitHub, Google ou e-mail.
  2. Crie uma nova pasta com o arquivo index.html e os recursos do site.
  3. No painel da Vercel, clique em “Add New...” > “Project”.
  4. Escolha “Import Git Repository” se seu projeto está no GitHub, ou use “Deploy” e faça upload da pasta.
  5. Vercel detecta sites estáticos automaticamente. Clique em “Deploy”.
  6. Em segundos seu site estará online com um link.

Alternativa rápida: GitHub Pages

Se quiser publicar pelo próprio GitHub, basta criar um repositório e subir o arquivo index.html na raiz.
Depois, nas configurações do repositório, ative o GitHub Pages e pronto.
